# 場地可借用時段查詢系統

臺北市立大學場地借用查詢系統，提供 G201、G202、G203、T201 四個會議室的可借用時段查詢。

## 網站功能

- 選擇場地查看未來 180 天的可借用時段
- 時段分為：早上 (08:00-12:00)、下午 (13:00-17:00)、晚上 (18:00-22:00)
- 顯示「可借用」或「已借用」狀態

| 場地代碼 | 名稱 | 容納人數 |
|----------|------|----------|
| G201 | 公誠樓第1會議廳 | 103 人 |
| G202 | 公誠樓第2會議廳 | 103 人 |
| G203 | 公誠樓第3會議廳 | 225 人 |
| T201 | T201會議室 | 50 人 |

## 部署

本系統為純靜態網站，透過 Zeabur 部署。

### 目錄結構

```
web/
├── index.html          # 主頁面
├── data/
│   └── availability.csv  # 可借用時段資料
├── css/
│   └── style.css       # 樣式表
└── js/
    └── app.js          # 前端邏輯
```
